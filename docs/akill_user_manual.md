# akill 用户手册

## 概述

`akill` 是 lsfMonitor 提供的命令行工具，用于批量终止（kill）LSF 作业。相比原生的 `bkill` 命令，`akill` 支持模糊匹配和多种筛选条件，可以更高效地批量管理作业。

## 使用方法

```
akill [选项]
```

## 命令行参数

| 参数 | 缩写 | 说明 |
|------|------|------|
| `--jobid` | `-j` | 根据 Job ID 终止作业，支持模糊匹配和范围 |
| `--job_name` | `-J` | 根据作业名终止作业，支持模糊匹配 |
| `--command` | `-c` | 根据命令终止作业，支持模糊匹配 |
| `--submit_time` | `-s` | 根据提交时间终止作业，支持模糊匹配 |
| `--execute_host` | `-m` | 根据执行主机终止作业 |
| `--queue` | `-q` | 根据队列终止作业 |
| `--user` | `-u` | 根据用户终止作业 |

所有参数均可接受多个值（以空格分隔）。

## 模糊匹配说明

`-j`、`-J`、`-c`、`-s` 参数支持使用 `*` 通配符进行模糊匹配，`*` 会被转换为正则表达式 `.*` 进行匹配。

## Job ID 特殊用法

- **精确 ID**：`akill -j 12345`
- **ID 范围**：`akill -j 10200-10450`（终止 10200 到 10450 之间的所有作业）
- **模糊匹配**：`akill -j 123*`（终止所有以 123 开头的作业）
- **终止全部**：`akill -j 0`（终止当前用户的所有作业）

## 使用示例

### 根据 Job ID 终止作业

```bash
# 终止单个作业
akill -j 12345

# 终止多个作业
akill -j 12345 12346 12347

# 终止一个范围内的作业
akill -j 10200-10450

# 模糊匹配终止
akill -j 123*
```

### 根据作业名终止

```bash
# 终止名为 my_job 的作业
akill -J my_job

# 模糊匹配
akill -J sim_*
```

### 根据命令终止

```bash
akill -c run_simulation*
```

### 根据提交时间终止

```bash
akill -s "Apr 15*"
```

### 根据执行主机终止

```bash
# 终止某主机上的所有作业
akill -m host01
```

### 根据队列终止

```bash
# 终止某队列中的所有作业
akill -q normal
```

### 根据用户终止

```bash
# 终止某用户的所有作业
akill -u user01
```

### 组合使用

```bash
# 终止用户 user01 在 normal 队列中 ID 为 10000-20000 的作业
akill -u user01 -q normal -j 10000-20000
```

## 注意事项

- `akill` 底层调用 LSF 的 `bkill` 命令，需要确保当前用户有权限终止目标作业。
- 使用 `-j 0` 会终止当前用户的所有 RUN 和 PEND 作业，请谨慎使用。
- `-j`、`-J`、`-c`、`-s` 会先查询所有 RUN 和 PEND 作业（`bjobs -r -p -UF`），然后逐一匹配终止。
- `-m`、`-q`、`-u` 直接调用 `bkill` 的对应参数，由 LSF 处理筛选。
